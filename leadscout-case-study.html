Case Study

Nalon LeadScout

Python + FastAPI lead generation dashboard built to help freelancers, web designers, digital marketers, and sales teams find local businesses with no listed websites.

Open Live Demo Request Demo

Tech Stack

Python FastAPI Uvicorn Jinja2 HTML CSS Pandas Requests OpenStreetMap Overpass API CSV Automation Google Sheets Gmail Draft Workflow

Overview

Nalon LeadScout is a lead generation automation tool built to help freelancers, web designers, digital marketers, and sales teams find local businesses with no listed websites.

The system uses free OpenStreetMap data through the Overpass API to search businesses by city and category, detect missing websites, score leads, generate outreach-ready messages, and export CSV files for sales follow-up.

Problem

Many small businesses are listed online but do not have a proper website. This creates an opportunity for freelancers, agencies, and service providers to offer websites, SEO, digital marketing, POS systems, or other business services.

The challenge is that finding these businesses manually takes too much time.

Solution

I built Nalon LeadScout as a web dashboard that allows users to select a city and business category, then automatically generate a list of businesses with no listed website.

The app filters and scores leads, creates Google Maps and Google Search verification links, prepares outreach messages, and exports clean CSV files for follow-up.

Key Features

  • Web dashboard built with FastAPI
  • City and business category selection
  • Uses free OpenStreetMap / Overpass API data
  • Detects businesses with no listed website
  • Generates Google Maps verification links
  • Generates Google Search verification links
  • Scores leads based on contact and location data
  • Labels leads by quality
  • Recommends best contact channel
  • Generates outreach-ready email messages
  • Removes duplicate businesses
  • Handles failed API servers using fallback servers

New Web App Features

  • Button-based dashboard instead of running scripts manually
  • Stats cards showing total leads, email-ready leads, high-quality leads, and leads with phone numbers
  • Download buttons for full leads, email-ready leads, and outreach messages CSV files
  • Lead results table with business name, category, city, phone, email, lead quality, score, best contact channel, maps link, and search link
  • Portfolio-ready interface for demos

Workflow

  1. User selects a city and business category.
  2. The app searches OpenStreetMap business data.
  3. Businesses with listed websites are filtered out.
  4. Remaining leads are cleaned and deduplicated.
  5. Each lead is scored based on available contact/location data.
  6. The system generates verification links and outreach messages.
  7. User downloads CSV files for outreach.
  8. Email-ready leads can be uploaded to Google Sheets.
  9. Gmail drafts can be created for review before sending.

Result

1,711 leads generated

In testing, LeadScout generated 1,711 South African business leads across Johannesburg, Cape Town, Durban, Pretoria, and Sandton.

The result is a working FastAPI web dashboard that turns lead generation into a simple button-based workflow.

Business Use Cases

  • Web designers can find businesses that need websites.
  • Digital marketing agencies can find local businesses with weak online presence.
  • POS/payment companies can find restaurants, salons, bakeries, car washes, and retail businesses.
  • Sales teams can generate niche-specific local business lists.
  • Freelancers can use the tool to find prospects and offer one-page websites or digital services.
